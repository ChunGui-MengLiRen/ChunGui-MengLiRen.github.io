表单

渲染

  • 输入框

设置v-model指令

  • 单选框

多个单选框需要设置value且v-model指向同一个变量

  • 多选框

多选框需要设置value属性且v-model指向同一个数组变量

  • 勾选框

v-model需要指向一个布尔变量

  • 下拉框

给select设置v-model，指向一个数组变量，option需要设置value属性

  • 文本域

设置v-model指令

  • 提交

取消默认提交行为

  • 重置

创建一个初始对象数据，给data中的数据拷贝一份，重置表单时重新还原初始对象的值

修饰符

  • number

转化为数值类型

  • trim

去除字符串两端空格

  • lazy

输入完才修改

自定义指令

  • 目的

解决内置指令无法满足需求

  • 使用

计算属性

  • 目的

本质为属性，为了使模板使用更加方便，对data中的属性进行加工处理

  • 使用
  • 注意
    • 计算属性一般不能修改
    • 计算属性只用当依赖的数据改变时才会重新执行

侦听器

  • 目的

    侦听属性发生改变

  • 使用

过滤器

  • 目的

对数据进行加工处理用来满足模板的使用，本质为工具函数

  • 使用

生命周期

创建

  • 创建前 beforeCreate

data数据未初始化，无法操作data数据

  • 创建后 created

data中数据完成初始化，可以操作data数据，此时可以请求数据

挂载

  • 挂载前 beforeMount

数据未渲染到模板中

  • 挂载后 mounted

数据已经渲染到模板，可以使用插件初始化

更新

  • 更新前 beforeUpdate

数据更新之前

  • 更新后 update

数据更新后，可以使用插件初始化

销毁

  • 销毁前 beforeDestory

Vue实例销毁前 vm.$destory() 销毁当前实例

  • 销毁后 destoryed

Vue实例销毁之后
